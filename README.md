# Oh My Dotfiles

Yet another dotfiles repo, boring

## Getting Started

### Operating system compatibility

| OS            | Status |
|:-------------:|:------:|
| macOS         |   ✅    |
| Linux         |   ❌    |
| Windows (WSL) |   ❌    |

### Prerequisites

- `curl` should be installed
- `git` should be installed (recommended `v2.7.0` or higher)

### Basic installation

```sh
/bin/sh -c "$(curl -fsSL https://raw.githubusercontent.com/ymokry/oh-my-dotfiles/HEAD/install.sh)"
```

_After installation, all config files will be located under `$HOME/.oh-my-dotfiles`_
